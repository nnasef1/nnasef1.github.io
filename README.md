# nnasef1.github.io

Personal site for Nariman Nasef — independent program leadership and executive coaching for engineering teams shipping mission-critical software.

Live at https://nnasef1.github.io.

## Local development

```sh
npm install        # one-time, installs puppeteer for screenshot tooling
node serve.mjs     # serves the project at http://localhost:3000
node screenshot.mjs http://localhost:3000 label --full   # full-page screenshot
```

`index.html` is a single-file Tailwind (CDN) build. Edit, refresh.
